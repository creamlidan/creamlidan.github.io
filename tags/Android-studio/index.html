adb server version (41) doesn't match this client (39); killing...

lidan 发表于 2019-05-31    标签： Android studio

- 链接不上夜神模拟器提示adb版本不符

解决方案

  • 先关闭所有 adb 的进程；
  • 将 SDK 的 platform-tools 目录下的 adb.exe 文件复制到电脑任意路径下，并改名为 nox_adb.exe 。然后再把改名后的文件拷贝到夜神模拟器安装路径的 bin 目录下替换原来的 nox_adb.exe 文件
  • 将 SDK 的 platform-tools 目录下的 adb.exe 替换夜神模拟器安装路径的 bin 目录下替换原来的adb.exe(夜神bin目录下有两个需要替换)
  • 如果还是版本不一致注意一下将 SDK 的 platform-tools 目录下的 adb.exe与将 SDK 的 tools下的adb版本是否一致
  • 重启模拟器。
  • 命令窗口输入adb devices查看设备信息
  • 连接夜神adb
  • 如果出现adb关闭,需要注意一下夜神的android版本必须切换到5以上,bin目录下打开多开器，点击添加模拟器下载android5模拟器，然后启动，在链接就ok了
